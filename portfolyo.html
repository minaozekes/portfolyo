İzmir Ekonomi Üniversitesi Meslek Yüksekokulu'nda Bilgisayar Programcılığı okuyorum. Python, JavaScript, React Native, C#, Java ve Arduino ile projeler geliştiriyorum; web uygulamalarından mobil geliştirmeye, veri biliminden gömülü sistemlere geniş bir yelpazede çalışıyorum.

Uzun vadeli hedefim yapay zekâyı otomotiv sistemlerine entegre ederek sağlam yazılımlar üretmek. Üniversitenin Yazılım Kulübü aktif üyesiyim; iki yıl üst üste Gameathon etkinliğinde mentörlük yaptım, Bahar Şenliği organizasyonunda görev aldım ve OBS Destek Masası (Ders Tercih Günleri)'nde öğrencilere destek verdim.
